# MFA Enforcement

## Objective

Implement Multi-Factor Authentication (MFA) to improve account security.

## Scope

- Administrative Accounts
- Standard Users
- Remote Access Users

## Implementation Steps

### Review User Accounts

- Identify active users
- Review privileged accounts

### Enable MFA

- Microsoft Authenticator
- SMS Authentication
- Authentication Methods Policy

### User Communication

- Notify users
- Provide enrollment instructions

## Validation

- MFA challenge successful
- User registration completed
- Authentication logs verified

## Benefits

- Reduced account compromise risk
- Improved identity protection

## Skills Demonstrated

- Microsoft 365 Administration
- Entra ID
- Identity Protection
- Security Hardening
