Exchange Online Security

Objective

Implement security controls to protect Exchange Online against phishing, spam, malware, and unauthorized access.

Environment

* Microsoft 365
* Exchange Online
* Microsoft Defender for Office 365
* Microsoft Entra ID

Security Controls

Anti-Spam Protection

Configured policies to:

* Block malicious senders
* Filter spam messages
* Reduce unwanted email traffic

Anti-Phishing Protection

Configured policies to:

* Detect impersonation attempts
* Protect executive accounts
* Monitor suspicious email activity

Malware Protection

Configured policies to:

* Scan attachments
* Block malicious files
* Prevent malware delivery

Mail Flow Security

Reviewed and validated:

* SPF records
* DKIM configuration
* DMARC policies

Administrative Security

Implemented:

* MFA for administrators
* Role-based access control
* Administrative activity monitoring

Validation

Verified:

* Email delivery functionality
* Anti-spam filtering
* Anti-phishing protection
* Authentication records
* Security alerts

Benefits

* Improved email security
* Reduced phishing attacks
* Better sender validation
* Stronger Microsoft 365 security posture

Skills Demonstrated

* Exchange Online Administration
* Email Security
* SPF
* DKIM
* DMARC
* Microsoft Defender
* Microsoft 365 Security
