# Microsoft 365 Security Baseline

This repository contains Microsoft 365 security hardening notes, best practices, and deployment procedures.

## Security Controls

### Identity Security
- Multi-Factor Authentication (MFA)
- Conditional Access Policies
- Password Policies
- Entra ID Security

### Email Security
- Exchange Online Protection
- SPF
- DKIM
- DMARC
- Anti-Phishing Policies

### Endpoint Security
- Microsoft Defender
- BitLocker
- Device Compliance

### Collaboration Security
- SharePoint Security
- Teams Security
- External Sharing Controls

## Documentation

This repository is used for lab work, documentation, and security reference materials.
